# Exploratory Analysis of European Soccer Leagues

This project explores match data from major European football leagues using R and data visualization techniques. The analysis focuses on goal-scoring patterns, league comparisons, home advantage, seasonal trends, and possession statistics.

## Files Included

- `ExerciseSheet_02.Rmd` – R Markdown source file
- Generated HTML report
- Supporting visualizations and analysis

## Dataset

European Soccer Database:
https://www.kaggle.com/datasets/hugomathien/soccer

## Project Report

View the complete report here:

https://rpubs.com/Gold166/european-football-eda

## Tools Used

- R
- ggplot2
- dplyr
- RMarkdown
- SQLite

## Key Insights

- Compared average goals across major European leagues.
- Investigated differences between top leagues and other leagues.
- Examined the home advantage effect.
- Analyzed seasonal goal-scoring trends.
- Evaluated the distribution of home team possession.
